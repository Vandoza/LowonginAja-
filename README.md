# Lowongin Aja!

Platform rekomendasi lowongan untuk **mahasiswa IF & STI HMIF ITB**.  

## Cara Mengakses

1. Buka **https://lowongin-aja.vercel.app**  
2. **Login** dengan:
   - Nama: `John Doe`
   - NIM: `18223000`
3. Isi **Data Diri**:
   - Minat
   - Keahlian
   - **Mata Kuliah** (boleh lebih dari satu):  
     - Teknologi Sistem Terintegrasi  
     - Pengembangan Aplikasi dan Web  
     - Arsitektur Enterprise
   - (Opsional) Projek kuliah
4. Klik **Lanjut ke Lowongan** → lihat daftar **lowongan yang relevan**.
5. Klik **Tambah** pada salah satu lowongan:
   - Pertama kali → muncul modal **“Selamat! Anda diterima!”**  
   - Jika mencoba melamar lagi → **“Anda sudah memiliki pekerjaan.”**

## Troubleshooting

- **Tidak dapat login**: pastikan gunakan `John Doe` dan `18223000`.  
- **Lowongan kosong**: coba **Refresh** (tombol *Muat Ulang*), atau cek koneksi internet / ad-blocker.
